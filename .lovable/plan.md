
# خطة بناء مشروع تكوين — المرحلة 1 و 2

## المرحلة 1: قاعدة البيانات + الإعدادات

### 1. ربط المشروع بـ Supabase
- تفعيل Supabase وربطه بالمشروع

### 2. إنشاء جداول قاعدة البيانات
- جدول **settings**: إعدادات الموقع (الهاتف، البريد، واتساب، العنوان، روابط التواصل، صور الهيرو، عداد الزيارات، كلمة مرور المدير)
- جدول **projects**: المشاريع (عنوان عربي/إنجليزي، الفئة، النوع، الصورة المصغرة، رابط الفيديو، الترتيب، مميز، مرئي)
- جدول **requests**: طلبات العملاء (الاسم، الجهة، نوع الخدمة، التاريخ، الموقع، التفاصيل، الهاتف، البريد، الحالة)

### 3. إعداد سياسات الأمان (RLS)
- قراءة عامة للإعدادات والمشاريع المرئية
- إدراج عام لطلبات العملاء
- صلاحيات كاملة للمستخدمين المسجلين (المدير)

### 4. إنشاء الملفات البرمجية
- `src/integrations/supabase/types.ts` — أنواع TypeScript للجداول
- `src/integrations/supabase/client.ts` — عميل Supabase
- `src/hooks/useSettings.ts` — هوك لجلب وتحديث الإعدادات
